JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons

  • Variable Declaration JavaScript variables are containers for storing data values- imagine a cup you fill with coffee, the cup holds the coffee, a varaible holds a value. All Java Script variables must be identified with unique names. These unique names are called identifiers.var x;
  • Variable Assignment Assignment operators assign values to JavaScript variables - our cup can now have coffee pourd in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x=10;
  • Data Types Data types are an important concept to be able to operate on variables you need to know the data type. There are six datatypes that are JavaScript Primities: Boolean - true or false; null - null aka nothing;
